Description: Toorg is the wisest member of the group of heroes called The Protectors of the Milky Way. For any question, it has the ideal answer! Write a program that, given a question, tells Toorg's answer.

Input: The input will have several test cases. For each test case, aNnumber is displayed, which represents the number of test cases. Then there will be N lines, with questions asked for Toorg.

Output: For each test case, print Toorg's answer.

Examples:
- Example 1:
  Input:
3
Who are you?
How old are you?
What can I do for you?
  Output:
I am Toorg!
I am Toorg!
I am Toorg!
