Description: In this problem we are interested in the frequency of letters in a given line of text.

Input: The input contains many test cases. The first line contains an integerNthat indicates the number of test cases. Each test case consists of a single line of text. The line may contain non-letter characters, but is guaranteed to contain at least one letter and less than 200 characters in total.

Output: For each test case, output a line containing the most frequently occurring letter(s) from the text in lowercase (if there are ties, output all such letters in alphabetical order).

Examples:
- Example 1:
  Input:
3
Computers account for only 5% of the country's commercial electricity consumption.
Input
frequency letters
  Output:
co
inptu
e
