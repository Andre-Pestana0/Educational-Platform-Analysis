Input: The input file contains 4 integer values.

Output: PrintDIFERENCA(DIFFERENCE in Portuguese) with all the capital letters, according to the following example, with a blank space before and after the equal signal.

Examples:
- Example 1:
  Input:
5
6
7
8
  Output:
DIFERENCA = -26

- Example 2:
  Input:
0
0
7
8
  Output:
DIFERENCA = -56

- Example 3:
  Input:
5
6
-7
8

DIFERENCA = 86
