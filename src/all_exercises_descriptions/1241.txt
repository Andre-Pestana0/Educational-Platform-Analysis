Input: The input consists of several test cases. The first line of input contains an integerNthat indicates the number of test cases. Each test case consists of two numbersAandBgreather than zero, with up to 1000 digits each.

Output: For each test case, print a message informing if the second number fit ("encaixa" in portughese) or didn't fit ("nao encaixa") in the first number.
