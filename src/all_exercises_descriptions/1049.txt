Description: In this problem, your job is to read three Portuguese words. These words define an animal according to the table below, from left to right. After, print the chosen animal defined by these three words.

Input: The input contains 3 words, one by line, that will be used to identify the animal, according to the above table, with all letters in lowercase.

Output: Print the animal name according to the given input.

Examples:
- Example 1:
  Input:
vertebrado
mamifero
onivoro
  Output:
homem

- Example 2:
  Input:
vertebrado
ave
carnivoro
  Output:
aguia

- Example 3:
  Input:
invertebrado
anelideo
onivoro
  Output:
minhoca
