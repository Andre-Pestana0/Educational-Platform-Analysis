Description: Make a program that reads a seller's name, his/her fixed salary and the sale's total made by himself/herself in the month (in money). Considering that this seller receives 15% over all products sold, write the final salary (total) of this seller at the end of the month , with two decimal places.

Input: The input file contains a text (employee's first name), and two double precision values, that are the seller's salary and the total value sold by him/her.

Output: Print the seller's total salary, according to the given example.

Examples:
- Example 1:
  Input:
JOAO
500.00
1230.30
  Output:
TOTAL = R$ 684.54

- Example 2:
  Input:
PEDRO
700.00
0.00
  Output:
TOTAL = R$ 700.00

- Example 3:
  Input:
MANGOJATA
1700.00
1230.50
  Output:
TOTAL = R$ 1884.58
