Description: Read an integer N. Print the square of each one of the even values from 1 to N including N if it is the case.

Input: The input contains an integer N (5 <N< 2000).

Output: Print the square of each one of the even values from 1 to N, as the given example.

Examples:
- Example 1:
  Input:
6
  Output:
2^2 = 4
4^2 = 16
6^2 = 36
