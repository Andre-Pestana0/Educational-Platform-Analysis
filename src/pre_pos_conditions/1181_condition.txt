Below is a **Hoare‑logic style specification** for the program that solves the problem.  
Only the *pre‑condition* (what must be true before the program starts) and the *post‑condition* (what must be true when the program terminates) are given – the body of the program itself is omitted, as requested.

### 1. Notation used  

* `ℤ` – set of integers, `ℝ` – set of real numbers.  
* `ℕ₀ = {0,1,2,…}` – non‑negative integers.  
* `M ∈ ℝ^{12×12}` – a 12 × 12 matrix of real numbers, indexed by rows `i` and columns `j` with `0 ≤ i,j ≤ 11`.  
* `L ∈ ℕ₀` – the *line* (row) supplied by the user.  
* `T ∈ {'S','M'}` – the operation code (`'S'` = **Sum**, `'M'` = **Average**).  
* `Σ(i=a..b) f(i)` – usual finite sum notation.  
* `⌊x⌉₁` – the value of `x` rounded to **exactly one digit after the decimal point** (the “print‑with‑one‑digit” format required by the statement).  
* `⟦P⟧` – the truth value of a predicate `P` (true = 1, false = 0).  

We also define the *green area* that the statement refers to.  
From the description and the illustrated example (the line number `L` determines a right‑angled triangular region whose legs lie on the top row and the leftmost column), the set of matrix positions that belong to the green area is

\[
G(L) \;=\; \{(i,j) \mid 0 \le i \le L,\; 0 \le j \le L\}.
\]

Thus the green area is the **upper‑left square sub‑matrix of size \((L+1) \times (L+1)\)**.  
The number of elements in that area is

\[
N(L) \;=\; |G(L)| \;=\; (L+1)^2 .
\]

The **sum** of all elements in the green area is

\[
S(L,M) \;=\; \sum_{i=0}^{L}\;\sum_{j=0}^{L} M[i][j].
\]

The **average** of those elements is

\[
A(L,M) \;=\; \frac{S(L,M)}{N(L)} .
\]

### 2. Hoare triple  

\[
\boxed{\;\{\,\text{Pre}\,\}\;\; \text{Program} \;\;\{\,\text{Post}\,\}\;}
\]

#### Pre‑condition  

\[
\begin{aligned}
\text{Pre} \;\equiv\; & \bigl( 0 \le L \le 11 \bigr) \;\land\;
                         \bigl( T \in \{'S','M'\} \bigr) \;\land\\[2mm]
                      & \bigl( M \in \mathbb{R}^{12\times12} \bigr).
\end{aligned}
\]

All three inputs are therefore guaranteed to be within their legal ranges before execution begins.

#### Post‑condition  

Let  

\[
\begin{aligned}
\text{result} &\;:=\; 
   \begin{cases}
      \displaystyle S(L,M)               &\text{if } T = 'S',\\[4mm]
      \displaystyle A(L,M) = \dfrac{S(L,M)}{N(L)} &\text{if } T = 'M'.
   \end{cases}
\end{aligned}
\]

The program must terminate **printing** a single real number `out` such that  

\[
\boxed{\; \text{out} \;=\; \big\lfloor \text{result} \big\rceil_{1}\;}
\]

i.e. `out` is `result` rounded to exactly one digit after the decimal point (the format required by the statement).  

Formally,

\[
\begin{aligned}
\text{Post} \;\equiv\;
   &\bigl( T = 'S' \;\rightarrow\; \text{out}= \big\lfloor S(L,M) \big\rceil_{1} \bigr) \;\land\\[2mm]
   &\bigl( T = 'M' \;\rightarrow\; \text{out}= \big\lfloor \tfrac{S(L,M)}{N(L)} \big\rceil_{1} \bigr).
\end{aligned}
\]
