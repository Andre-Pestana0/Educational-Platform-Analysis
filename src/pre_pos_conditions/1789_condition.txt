Only the **pre‚ÄĎcondition** (what must hold before the program starts) and the **post‚ÄĎcondition** (what is guaranteed after it finishes) are given; the body of the program (the actual code) is omitted ‚Äď any implementation that respects the two conditions is correct.

### 2.   The **pre‚ÄĎcondition**  

For each test case the input must satisfy  

\[
\boxed{
P \;\; \equiv\;\;
\bigl(L\in\mathbb N^{+}\;\wedge\;1\le L\le 500\bigr)
\;\wedge\;
\Bigl(\forall i\in\{1,\dots ,L\}\;\bigl(V_i\in\mathbb N\;\wedge\;1\le V_i\le 50\bigr)\Bigr)
}
\]

In words:  

* `L` is a positive integer not larger than 500;  
* each speed `V_i` is an integer between 1 and 50 (inclusive).

---

### 3.   The **post‚ÄĎcondition**  

Let  

\[
M \;:=\; \max_{1\le i\le L} V_i
\]

and define the **level function**  

\[
\mathsf{level}(m) \;:=\;
\begin{cases}
1 & \text{if } m < 10,\\[4pt]
2 & \text{if } 10 \le m < 20,\\[4pt]
3 & \text{if } m \ge 20 .
\end{cases}
\tag{1}
\]

The program must output exactly the value  

\[
\boxed{
Q\;\; \equiv\;\; \text{output} = \mathsf{level}(M)
}
\]

i.e. the level that corresponds to the **fastest** slug of the group.

---

### 4.   Hoare triple for a **single test case**

\[
\boxed{\{\,P\,\}\;\; \text{slug\_level\_program}\;\; \{\,Q\,\}}
\]

where `slug_level_program` is any sequence of statements that

* reads `L` and the vector `V`,
* computes `M = max_i V_i`,
* evaluates the case distinction (1) and prints the resulting `‚Ąď`.

---

### 5.   Hoare triple for the **whole input stream**  

The program repeatedly executes the above triple for every test case until EOF.  
Let  

\[
\text{EOF} \;:=\; \text{no more input lines}
\]

and let `P_i` / `Q_i` denote the pre‚ÄĎ and post‚ÄĎconditions for the *i‚ÄĎth* test case.  
The overall specification can be written as

\[
\boxed{
\bigl(\forall i\ge 1\;\bigl(P_i \rightarrow Q_i\bigr)\bigr)
\;\wedge\;
\text{termination on EOF}
}
\]

which reads: *for each test case the program, assuming the input satisfies the pre‚ÄĎcondition `P_i`, terminates and guarantees the post‚ÄĎcondition `Q_i`*.

### 6.   Summary (compact form)

\[
\begin{aligned}
\{ & L\in\mathbb N^{+}\wedge 1\le L\le 500 \\
   & \wedge\;\forall i\in\{1,\dots ,L\}\;(V_i\in\mathbb N\wedge 1\le V_i\le 50) \;\}

   \;}

Any implementation that respects the above Hoare triple is a correct solution to the ‚Äúfast‚ÄĎslug level‚ÄĚ problem.
