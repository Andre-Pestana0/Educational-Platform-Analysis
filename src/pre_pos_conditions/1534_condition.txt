**Hoare‑style specification**

We describe two units:

*`ReadAndPrint`* – the whole program that repeatedly reads test cases until end‑of‑file.  
*`PrintMatrix(N)`* – the routine that, given a single size `N`, prints the required
\(N\times N\) array.

---

### 1.  `PrintMatrix(N)`

```
{ 3 ≤ N < 70 }                     ──  Pre‑condition
PrintMatrix(N)
{  ∀ i,j ∈ {1,…,N} :
        Out(i,j) = 
        ┌ '2'  if  i + j = N + 1
        │ '1'  if  i = j  ∧  i + j ≠ N + 1
        └ '3'  otherwise
   ∧  the output consists of N lines,
      line i being   Out(i,1) Out(i,2) … Out(i,N)  followed by ‘\n’ }
```

*Explanation of the post‑condition*

* `Out(i,j)` denotes the character that is printed at row `i` and column `j`.  
* The three cases are mutually exclusive:

  1. **Anti‑diagonal** – if the cell lies on the anti‑diagonal  
     \(i + j = N + 1\) the digit **2** is printed.  
  2. **Main diagonal (but not anti‑diagonal)** – if the cell lies on the main
     diagonal \(i = j\) **and** it is not simultaneously on the anti‑diagonal,
     the digit **1** is printed.  
  3. **All other cells** – digit **3** is printed.

* The output format is exactly `N` lines, each line containing `N` characters
  (no spaces) and terminated by a line‑feed character.

---

### 2.  `ReadAndPrint`

Let the input stream be a (possibly empty) finite sequence  

\[
\langle N_1 , N_2 , \dots , N_k\rangle ,\qquad k\ge 0,
\]

where each element satisfies the same size restriction.

```
{  Input = ⟨ N1 , … , Nk ⟩  with  ∀t ( 3 ≤ Nt < 70 ) }    ──  Pre‑condition
ReadAndPrint
{  Output =  Π_{t=1}^{k}   PrintMatrix(Nt) }               ──  Post‑condition
```

* `Π` denotes concatenation of the strings produced by the successive calls
  to `PrintMatrix`.  
* Consequently the whole program writes, in the order the numbers appear in the
  input, the matrices defined by the specification of `PrintMatrix(N)`.

---

### 3.  Summary in a single Hoare triple

\[
\boxed{
\{\,\text{input is a (possibly empty) list } \langle N_1,\dots,N_k\rangle
   \text{ with } 3\le N_t<70\,\}
\;\; \textbf{ReadAndPrint}
\;\;
\{\,\text{output} = \bigcup_{t=1}^{k}
   \text{matrix}(N_t)\, \}
}
\]

where  

\[
\text{matrix}(N)[i,j] =
\begin{cases}
'2' & \text{if } i+j = N+1,\\[2mm]
'1' & \text{if } i=j\ \land\ i+j\neq N+1,\\[2mm]
'3' & \text{otherwise},
\end{cases}
\qquad 1\le i,j\le N .
\]

This formalisation captures **all** the requirements of the problem:
* the admissible range of `N`,
* the termination condition (EOF),
* the exact digit that must appear in every cell,
* and the exact layout of the printed result.
